This directory contains IDE projects, source code files, etc. for assemblers, assembler generators,
disassemblers, assembler test programs, unit tests and a framework to build and extend all of the above.

We have provided IDE project files for NetBeans and Eclipse, but no ant build files.

The addressed ISAs are: SPARC, AMD64, IA32 and PowerPC.
